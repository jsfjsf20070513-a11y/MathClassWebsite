# 数学课程班级网站

这是一个基于 React 和 Vite 构建的班级网站，用于展示数学课程的活动、通知和资源。

## 前置要求

你需要安装 Node.js 才能运行此项目。请访问 [nodejs.org](https://nodejs.org/) 下载并安装。

## 快速开始

1. **安装依赖**

   打开终端，运行以下命令：
   ```bash
   npm install
   ```

2. **启动开发服务器**

   依赖安装完成后，运行：
   ```bash
   npm run dev
   ```

   然后在浏览器中访问控制台输出的地址（通常是 http://localhost:5173）。

## 功能

- 展示近期班级活动
- 列出常用学习资源
- 响应式设计

## 技术栈

- React
- Vite
